# テーブル [(Tables)](https://hacknock.github.io/lobster-wiki/?page=example-table)

lobster.js は、標準の Markdown テーブルに加え、**セルの結合**と、レイアウトグリッド用の**サイレントテーブル**という 2 つの強力な拡張機能をサポートしています。

## 標準テーブル

```markdown
| 名前        | 型      | 説明                                |
| :---------- | :------ | :---------------------------------- |
| `src`       | string  | Markdown ファイルへのパスまたは URL |
| `container` | Element | ターゲットの DOM 要素               |
| returns     | Promise | レンダリングが完了した時に解決      |
```

**結果:**

| 名前        | 型      | 説明                                |
| :---------- | :------ | :---------------------------------- |
| `src`       | string  | Markdown ファイルへのパスまたは URL |
| `container` | Element | ターゲットの DOM 要素               |
| returns     | Promise | レンダリングが完了した時に解決      |

## 列の配置

区切り行で `:---`、`:---:`、および `---:` を使用します:

```markdown
| 左寄せ | 中央寄せ |     右寄せ |
| :----- | :------: | ---------: |
| りんご |  バナナ  | さくらんぼ |
| 1      |    2     |          3 |
```

**結果:**

| 左寄せ | 中央寄せ |     右寄せ |
| :----- | :------: | ---------: |
| りんご |  バナナ  | さくらんぼ |
| 1      |    2     |          3 |

## セルの結合

### 水平方向の結合

セルの内容に `\|` を追加すると、右側の 1 列分拡張されます。**ヘッダー行**と**本文行**の両方で機能します。`\|` セルの直後のスロットは結合によって消費されます:

```markdown
| 製品  | EMEA \| | APAC \| |
| :---- | ------: | ------: | --: | --: |
|       |      Q1 |      Q2 |  Q1 |  Q2 |
| Alpha |     100 |     120 |  80 |  90 |
| Beta  |      95 |     115 |  75 |  85 |
```

**結果:**

| 製品  | EMEA \| | APAC \| |
| :---- | ------: | ------: | --: | --: |
|       |      Q1 |      Q2 |  Q1 |  Q2 |
| Alpha |     100 |     120 |  80 |  90 |
| Beta  |      95 |     115 |  75 |  85 |

本文行のみの例:

```markdown
| 機能           | lobster.js | 標準 MD |
| :------------- | :--------: | :-----: |
| テーブル       |     ✓      |    ✓    |
| セル結合       | ✓ のみ \|  |
| ワープブロック | ✓ のみ \|  |
```

**結果:**

| 機能           | lobster.js | 標準 MD |
| :------------- | :--------: | :-----: |
| テーブル       |     ✓      |    ✓    |
| セル結合       | ✓ のみ \|  |
| ワープブロック | ✓ のみ \|  |

### 垂直方向の結合

セル内に `\---` と記述すると、上のセルと結合されます:

```markdown
| ブラウザ | エンジン |
| :------- | :------- |
| Chrome   | Blink    |
| Edge     | \---     |
| Firefox  | Gecko    |

| Safari | WebKit |
```

**結果:**

| ブラウザ | エンジン |
| :------- | :------- |
| Chrome   | Blink    |
| Edge     | \---     |
| Firefox  | Gecko    |
| Safari   | WebKit   |

### 結合の組み合わせ

```markdown
| 地域       | Q1     | Q2  |
| :--------- | :----- | :-- |
| ヨーロッパ | 120    | 150 |
| \---       | 130 \| |
| アジア     | 200    | 180 |
```

**結果:**

| 地域       | Q1     | Q2  |
| :--------- | :----- | :-- |
| ヨーロッパ | 120    | 150 |
| \---       | 130 \| |
| アジア     | 200    | 180 |

## サイレントテーブル(レイアウトグリッド)

各行の先頭に `~ ` を付けると、境界線のないレイアウトグリッドが作成されます。左右に並べて比較するのに最適です:

```markdown
~ | 左カラム | 右カラム |
~ | :--- | :--- |
~ | 左側のコンテンツです。| 右側のコンテンツです。|
~ | さらに左のコンテンツ。| さらに右のコンテンツ。|
```

**結果:**

~ | 左カラム | 右カラム |
~ | :--- | :--- |
~ | 左側のコンテンツです。| 右側のコンテンツです。|
~ | さらに左のコンテンツ。| さらに右のコンテンツ。|

サイレントテーブルは、**ワープ(Warp)** を使用したマルチカラムレイアウトの構成要素です — 詳細については、[Warp / Multi-column](?page=lobster-wiki/demo-ja/example-warp-ja) ページを参照してください。
