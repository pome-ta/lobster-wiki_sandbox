# customize top

使い勝手は（なるべく）そのままに、`index.html` のscript タグに収まる程度に追加。


- [injection](?page=customize/injection)
  - html やスクリプトを埋め込む
    - 複数方法で実装可能
      - コードブロック内に直接書き込み
      - ファイル参照
- [fileSourceCode](?page=customize/fileSourceCode)
  - 指定ファイルのソースコードを描画
  - 言語指定可（シンタックスハイライトを反映させる設定であれば）


:::details `index.html` のソースコードを見る

```html:./../index.html

```

:::
